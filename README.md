# NNS_SDK_CSharp
a NNS contract function SDK for .net framwork and .net core

## Support for 

### .NET Framework 4.6.1+

### .NET Core 2.0+


## NuGet
[Nuget Page](https://www.nuget.org/packages/NNS.sdk/)

### Package Manager

```
Install-Package NNS.sdk -Version 1.0.2
```

### .NET CLI


```
dotnet add package NNS.sdk --version 1.0.2
```
